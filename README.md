# Features
- Compile jade in to html

# Usage
To start server:
```
npm start
```
and go to [http://localhost:3001](http://localhost:3001)


To auto compile `.scss` and `.js` from `src` to `assets` folder:
```
grunt watch
```

To convert jade into html:
```
grunt jade
```
